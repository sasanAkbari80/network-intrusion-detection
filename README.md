# Network Intrusion Detection with Machine Learning

A machine learning project that classifies network traffic as 
normal or malicious using the NSL-KDD dataset.


## Project Overview
- **Domain:** Network Security / Cybersecurity
- **Dataset:** NSL-KDD (Kaggle)
- **Goal:** Binary classification — normal traffic vs attack traffic
- **Models:** Logistic Regression, Random Forest

## Tools and Libraries
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter
